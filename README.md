# 泰坦尼克号生存预测

## 项目简介
本仓库包含"泰坦尼克号生存预测"机器学习案例的完整实现代码，包括数据预处理、特征工程、模型训练与评估。

## 快速开始
1. 克隆本仓库：`git clone https://github.com/fsrrr/titanic.git`
2. 安装依赖：`pip install -r requirements.txt`
3. 运行代码：`python titanic_analysis.py`

## 文件说明
- `titanic_analysis.ipynb`：完整的Jupyter Notebook分析流程
- `titanic_analysis.py`：可执行的Python脚本
- `requirements.txt`：项目依赖库
- `data/`：数据集文件夹
